# Hornbill
An Offline-First Emergency Response & Navigation Ecosystem
---
Description:
The Hornbill App introduces an offline-first emergency response ecosystem designed to bridge the communication gap prevalent in rural ASEAN region. Using peer-to-peer networking and built-in AI models, Hornbill is able to close the gap on emergency rescue times and improve disaster resilience in rural regions.
